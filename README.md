# Frontend-Mentor-recipe-page

A clean and responsive recipe page built as part of a Frontend Mentor challenge.
The project focuses on semantic HTML, responsive layout, and modern CSS practices.

---

## Built With

* HTML5
* CSS3
* Flexbox
* Responsive Design (Mobile-first approach)
* Google Fonts

---

## Features

* Responsive layout for mobile and desktop
* Styled lists and custom markers
* Semantic HTML structure
* Accessible and scalable CSS using `rem` units

---

## What I Learned

* How to replace `px` with scalable units like `rem`
* Centering layouts using Flexbox
* Building responsive designs using `max-width`
* Improving typography and spacing for better UI
* Structuring clean and maintainable CSS

---

## Challenges

* Making the layout responsive without using fixed widths
* Matching the design spacing accurately
* Handling list markers styling (`::marker`)
